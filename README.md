# Spil_udvikling

Et 2D top-down pixel-art indie game prototype bygget i Unity 6 LTS.

## Status

Den nuværende prototype indeholder:

- Player movement med Unity Input System
- Camera follow
- Synlig placeholder world
- NPC interaction og dialogue
- Basic combat
- Enemy AI
- Loot pickup
- Inventory coins/items
- Quest loop med Mira
- Simple shop NPC
- Save/load med F5/F9

## Sådan prøver du spillet

Hvis Windows buildet findes lokalt:

```bat
PLAY_NYT_SPIL.cmd
```

Hvis buildet ikke findes endnu, åbn projektet i Unity og vælg:

```text
Tools > Build > Build Windows Demo
```

Build output:

```text
Builds/Windows/NytSpil.exe
```

## Controls

- Move: WASD / piletaster
- Attack: Space / venstre mus
- Interact: E
- Save: F5
- Load: F9

## Unity Version

Unity `6000.3.15f1`
