# Visual Direction

Target style: polished cozy-mysterious pixel art with warm sunset light, magenta skies, lavender mountain shadows, deep pine greens, warm lanterns, stone paths, and small alpine village details.

Current visual anchor:

`Assets/_Project/Art/Backgrounds/mountain_village_playfield.png`

Prompt used:

> High-quality top-down pixel-art village clearing for a cozy but mysterious indie game. Warm sunset mountain palette, pink-purple sky glow, blue-violet shadows, golden highlights, alpine village atmosphere. Top-down playable area with grassy meadow, stone paths, old shrine, lantern posts, dark pine edges, distant purple mountains. No text, no watermark, no characters.

Implementation notes:

- Keep gameplay readability first: player, enemies, interactables and loot must contrast clearly against the detailed background.
- Use crisp point filtering and no texture compression for pixel-art source assets.
- Future character sprites should use the same sunset palette, but with brighter silhouettes than the environment.
- Avoid muddy one-hue palettes; this game should combine warm lantern/pink/gold highlights with cool violet/teal shadows.
